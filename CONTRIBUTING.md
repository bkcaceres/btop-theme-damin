# Contributing

Thank you for your interest — really.

This is a personal dotfiles repository, tuned for one author's setup. To keep it consistent with [`miniex/dotfiles.tmux`](https://github.com/miniex/dotfiles.tmux), [`miniex/dotfiles.kitty`](https://github.com/miniex/dotfiles.kitty), [`miniex/dotfiles.nvim`](https://github.com/miniex/dotfiles.nvim), and [`miniex/fish-theme-damin`](https://github.com/miniex/fish-theme-damin), **outside contributions are not accepted**. Pull requests will be politely closed; issues that aren't bug reports for the published behavior will be closed as well.

You are very welcome to:

- **Fork it.** The repo is MIT-licensed — copy what's useful, change anything you like, and run with it.
- **Open an issue** if something is plainly broken (`install.sh` crashes, btop refuses to load the theme, a documented theme key is missing or unreadable). Please include your OS, `btop --version`, and the exact command or steps.

For feature requests, palette tweaks, or new color variants — please keep them in your fork. I won't be merging them here.

Thanks again for understanding.

— Han Damin
